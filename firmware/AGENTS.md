# Project Instructions

## 项目背景
这是一个机器人项目，目标平台是 STM32G474VCT6。

## 构建方式
- 外设代码生成使用STM32CubeMX，不要改动 .ioc 配置
- Windows11 本地使用 MDK-ARM 及 VS code + ARM-pack进行开发

## 代码规则
- C 代码保持现有命名风格
- 这是一个多人协作项目，涉及环境的代码要具有普适性

## 验证要求
每次改完代码后：
1. 运行格式检查
2. 进行静态检查，无需编译

## 我的偏好
- 我提出需求后，与我讨论需求，我明确要求行动后再改代码
- 改动代码要基于当前项目代码风格
- 不要自动提交 git commit，除非我明确要求
